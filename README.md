# .ei

Welcome to the **.ei** repository, an application focused on the centralized management of reward points and loyalty programs (Livelo, Esfera, Dotz, among others).

This project was built to solve the problem of tracking expired points and managing miles, providing a scalable and secure RESTful API.

## 🚀 Technologies Used

The project ecosystem was designed with a focus on performance and market best practices:

- **Language & Framework:** C# with .NET
- **Database:** PostgreSQL
- **ORM & Versioning:** Entity Framework Core (Code-First)
- **Infrastructure Orchestration:** Docker and Docker Compose
- **Monitoring & Observability:** Serilog integrated with Seq
- **Security:** Authentication and Authorization via JWT (JSON Web Token)

## ⚙️ Prerequisites

To run the project locally, you will need the following tools installed on your machine:

- [.NET SDK](https://dotnet.microsoft.com/download) (latest version)
- [Docker Desktop](https://www.docker.com/products/docker-desktop)
- [Git](https://git-scm.com/)
